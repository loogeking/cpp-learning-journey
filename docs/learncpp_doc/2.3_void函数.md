# 2.3 — void 函数（无返回值的函数）

---

在前面的 [2.1 — 函数简介](2.1_函数简介.md) 中我们提到过，函数定义的基本语法长这样：

```cpp
returnType identifier() // 把 identifier 替换成你的函数名
{
    // 你的代码写在这里
}
```

虽然我们之前用过返回类型为 `void` 的函数，但并没有解释 `void` 到底是什么意思。本节就来专门讨论**返回类型为 `void`** 的函数。

---

## `void` 返回类型

**函数并不一定要向调用方返回值。** 为了告诉编译器“这个函数不返回任何值”，我们把返回类型写作 `void`。例如：

```cpp
#include <iostream>

// void 表示：这个函数不向调用方返回任何值
void printHi()
{
    std::cout << "Hi" << '\n';

    // 该函数没有返回值，所以不需要 return 语句
}

int main()
{
    printHi(); // 合法：调用了 printHi()，但没有返回值

    return 0;
}
```

在上面的例子里，`printHi` 有一个有用的行为（打印 “Hi”），但它**不需要**向调用方返回任何东西。因此它的返回类型被定义为 `void`。

当 `main` 调用 `printHi` 时，`printHi` 里的代码开始执行，打印 “Hi”。`printHi` 结束后，控制流回到 `main`，程序继续往下走。

**不返回任何值的函数**被称为 **非返回值函数（non-value returning function）**，也叫 **void 函数（void function）**。

---

## void 函数不需要写 `return` 语句

**void 函数在执行到函数末尾时会自动返回给调用方**，**不需要写 `return` 语句**。

你也可以在 void 函数里写一个**不带返回值的 `return` 语句**——这会让函数在执行到 `return` 处**提前返回**给调用方。这本来就是函数末尾自然要发生的事，所以**在 void 函数末尾写一个空的 `return;` 是多余的**：

```cpp
#include <iostream>

// void 表示这个函数不返回任何值
void printHi()
{
    std::cout << "Hi" << '\n';

    return; // 告诉编译器返回调用方——这句是多余的，因为函数末尾本来就会返回！
} // 函数在这里返回调用方

int main()
{
    printHi();

    return 0;
}
```

> **最佳实践**
>
> **不要在非返回值函数的末尾写 `return;`。**

---

## void 函数不能用在“需要值”的表达式中

某些表达式**必须**提供一个值。例如：

```cpp
#include <iostream>

int main()
{
    std::cout << 5; // 合法：5 是字面量，被送去控制台打印
    std::cout << ;  // 编译错误：没有提供值

    return 0;
}
```

`std::cout <<` 的右边必须是一个可以打印的值；如果什么都不给，编译器就会报语法错误。上面第二次调用 `std::cout` 没提供值，就编译不过。

再看下面这段：

```cpp
#include <iostream>

// void：这个函数不返回值
void printHi()
{
    std::cout << "Hi" << '\n';
}

int main()
{
    printHi();              // 合法：调用 printHi()，不需要返回值
    std::cout << printHi(); // 编译错误

    return 0;
}
```

- 第一次调用 `printHi()` 出现在**不需要值**的上下文中，因为这个函数本身也不返回值，所以没问题；
- 第二次调用 `printHi()` **根本编译不过去**。`printHi` 的返回类型是 `void`（不返回任何值），但这里却试图把它的“返回值”交给 `std::cout` 去打印。`std::cout` 不知道该拿它怎么办（它到底该输出个什么？）。所以编译器直接报错。**你需要注释掉这一行才能编译通过。**

> **小贴士**
>
> 有些语句**要求**你给它一个值，有些则**不要求**。
>
> - 当一条语句**只是一个函数调用**（比如上例中第一次的 `printHi()`）时，我们调用这个函数是**为了它的行为**，而不是它的返回值。这种上下文既可以调用**非返回值函数**，也可以调用**有返回值的函数并忽略其返回值**；
> - 当我们在**需要值的上下文**中调用函数时（比如 `std::cout <<` 的右边），**就必须提供一个值**。这种上下文下**只能调用有返回值的函数**。

```cpp
#include <iostream>

// 不返回值的函数
void returnNothing()
{
}

// 返回值的函数
int returnFive()
{
    return 5;
}

int main()
{
    // 当函数调用作为独立语句时，无需提供值
    returnNothing(); // 合法：可以调用一个不返回值的函数
    returnFive();    // 合法：也可以调用一个返回值的函数，并忽略它的返回值

    // 当在需要值的上下文（例如 std::cout）中调用函数时
    std::cout << returnFive();    // 合法：函数有返回值，可以被使用
    std::cout << returnNothing(); // 编译错误：这里不能调用返回 void 的函数

    return 0;
}
```

## void 函数如果试图返回值，会编译错误

**在非返回值函数里试图返回一个值**，会导致编译错误：

```cpp
void printHi() // 这是一个非返回值函数
{
    std::cout << "In printHi()" << '\n';

    return 5; // 编译错误：我们不应该在这里返回任何值
}
```
